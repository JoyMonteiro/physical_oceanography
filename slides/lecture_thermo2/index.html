Equation of State

$p=\rho RT$
$\alpha=\alpha_0(1 + \beta_T(T - T_0))$
$\alpha=\alpha_0(1 + \beta_T(T-T_0) - \beta_S(S - S_0) - \beta_p(p-p_0))$
$\alpha=\alpha_0(1 - \beta_T(1 + \gamma^* p)(T-T_0) + \frac{\beta^*}{2}(T - T_0)^2 - \beta_S(S - S_0) - \beta_p(p-p_0))$

Thermodynamic Relations

Extensive vs. Intensive
$I = I(\eta, \alpha, S)$
$dI = \overline{d}Q + \overline{d}W + \overline{d}C$
\[ dI = Td\eta - pd\alpha +\mu dS\\ T = \left (\frac{\partial I}{\partial \eta}\right )_{\alpha, S}\]

Thermodynamic Potentials

\[ h = I + p\alpha\\ dh = Td\eta + \alpha dp + \mu dS \]
\[ dg = -\eta dT + \alpha dp + \mu dS\\ df = -\eta dT + pd\alpha + \mu dS\\ dI = Td\eta - pd\alpha +\mu dS \]

Specific Heats

\[c_v = T\left (\frac{\partial\eta}{\partial T} \right )_{\alpha, S} = \left (\frac{\partial I}{\partial T} \right )_{\alpha, S}\\ c_p = T\left (\frac{\partial\eta}{\partial T} \right )_{p, S} = \left (\frac{\partial h}{\partial T} \right )_{p, S} \]
